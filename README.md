# ライブ配信でのチート利用について

## 原則禁止（安全最優先）

### 1. 対人戦・MOD・ツール利用
- 対人戦・ランキング・マルチプレイでは、公式以外のMODや外部ツールの使用、導入方法の共有、動画・配信での利用も禁止。
- 公式が明示して許可したもの以外は、閲覧・使用・共有すべてNG。

### 2. 公式ツールの条件
- 公式提供・公式許可の範囲内のみ使用可。
- 公式ツールでも、他プレイヤーへの優位性取得や用途逸脱はNG。

### 3. 相談・確認
- 使用や配信前に管理側へ必ず確認。
- 不明な場合は、配信・共有・導入を控える。

### 4. 心理的安全性
- 他人のチート行為や違反行為の晒しは禁止。
- 注意・是正は個別に行い、公開議論は避ける。

# 公式ツール利用ルール

## ① 「公式ツール」の定義

**公式ツールとは：**
- ゲーム運営・開発元が自ら提供している、または利用を明確に許可しているツール・機能

**公式ツールに含まれる例：**
- ゲーム内標準機能（設定・UIカスタマイズ等）
- 公式MOD／公式プラグイン
- 運営が「使用可」と明記している外部ツール
- 公式大会・競技シーンで使用が認められているツール

## ② 公式ツールでもNGになるケース

**以下はNG：**
- 対人戦以外向けとして提供されているものを、対人戦で使用する
- 本来の用途を逸脱して優位性を得る行為
- 運営が「条件付き許可」しているのに条件を満たしていない場合

## ③ グレーゾーンの裁定ルール
**判断基準（シンプル）：**
- 公式に「使用可」と明記されているか
- 対人戦・ランキングでの使用が許可されているか
- 使用することで他プレイヤーとの差が生じないか

→ 1つでも満たさなければNG

## ④ ルール
🎮 **対人戦におけるツール利用について**
- 対人戦・ランキング・マルチプレイにおいては、ゲーム運営・開発元が公式に提供、または使用を明確に許可しているツールのみ使用可能とします。
- 公式ツールであっても、対人戦での使用が想定・許可されていないもの、または本来の用途を逸脱して優位性を得る行為は禁止します。
- 使用可否が不明な場合は、配信・共有を行わず、事前に管理側へ相談してください。

## ⑤ 補足
本ルールはプレイヤーを責めるためのものではなく、コミュニティ内の公平性と安心感を守るためのものです。


# 草神くんより
## 1. 改造ハード・特殊操作によるRTA動画

例: スプラトゥーン1で海外の走者が日本語化の改造をしてRTAした映像。

論点: 「改造自体はコミュニティ的に認められているが、公式視点では違法行為」

**視聴の可否**

コミュニティルール視点:
RTAコミュニティで「公式大会以外の動画」として認められている場合は、観賞用として画面共有すること自体は問題視されにくい。

公式・法的視点:
ハード改造やROM改変は著作権・利用規約違反になる場合があるため、画面共有で広く配信するのは グレー～違法に近い扱い になる可能性あり。

💡 まとめ: 個人での視聴・学習目的であればコミュニティ的には許容されやすいが、他人に共有するとリスクがある。

## 2. 過去の3DS RTA動画や画面共有

- 管理の見解: 「3DSの画面共有（ハード改造も含め）自体ダメ」
- 重要な違い: 実際に自分の3DSを改造してプレイするのは明確にNG。
- 動画・過去の映像視聴:すでに公開された動画を 画面共有で見るだけ → これは基本的に著作権法の「私的使用の範囲」外になる可能性がある。
- 限定されたコミュニティ内で個人視聴的に見る分にはグレー。
- 公開配信（YouTubeなど）に近い形で画面共有するとアウト。

💡 ポイント: 「自分の勉強・分析用に見る」ならコミュニティ内で黙認されやすいが、広く配信する形はリスク。

## 3. レトロフリーク・ポリメガなどの互換機

**互換機の特徴: 正規ROMを挿入して遊ぶハード。コピーROMはNG。**

- 画面共有の可否:
- 正規ROMを互換機で動かす場合 → 法的にはグレーだけど、違法ではない可能性が高い
- コピーROMを使う場合 → 完全に違法、画面共有もアウト

💡 感覚的には:
「正規ROM＋互換機＋個人視聴」は比較的安全。「改造ROMや改造ハード」は注意が必要。
